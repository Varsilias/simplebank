### Create Local Environment File

Run `make env` a new file called `.env` will be created for you, fill in the values for enviroment variables

### Start Database Server

Run `make startdb`

### Migrate Database

Run `make migrate`

### Rollback Database

Run `make rollback`

### Run Tests With Coverage

Run `make test`
